# Contact Us Form

This is a simple **Contact Us Form UI** built using **HTML and CSS**.

## Features

* Clean Contact Form UI
* Contact Details Section
* Input Fields with Focus Effects
* Hover Effect on Button
* Responsive Layout using Flexbox
* CSS Pseudo-element Border Design

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Font Awesome Icons


## Responsive Design

The layout becomes **column-based on smaller screens** using a media query.

```
@media (max-width:700px)
```

This  site is live at https://mohsinkhan4050.github.io/Contact-Us-Form/
